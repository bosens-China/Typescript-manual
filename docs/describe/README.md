# 前言

JavaScript 从最初为网页添加交互，发展到至今可以跨平台开发，重要性不言而喻。
而作为一门动态语言，在编写大型项目的时候难免力不从心。动态语言就像是一把双刃剑，让你享受简洁语法的同时也会被其所限制。比如你要的是牛肉，但是动态语言只有你吃到嘴里才能真正知道是不是牛肉。

TypeScript 作为 JavaScript 的超集，为其提供了静态检查。如果你想为飞奔的野马套上缰绳，从现在起就开始学习吧。

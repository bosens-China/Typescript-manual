完整的破坏性改动列表请到这里查看:breaking change issues (opens new window).

# 标准库里的 DOM API 变动

  • 现在标准库里有 Window.fetch 的声明；仍依赖于 @types\whatwg-fetch 会产生声明冲突错误，需要被移除。
  • 现在标准库里有 ServiceWorker 的声明；仍依赖于 @types\service_worker_api 会产生声明冲突错误，需要被移除。
